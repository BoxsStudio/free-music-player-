# Free Music Player

Десктопный аудиоплеер на Electron с поддержкой HitMoZ и SoundCloud.

## Возможности

- Поиск и воспроизведение треков с HitMoZ (rus.hitmoz.org) и SoundCloud
- «Моя волна» — персональные рекомендации на основе прослушиваний и избранных треков
- Избранное — сохранение треков в localStorage
- Полноэкранный режим «Сейчас играет» с настройкой фона
- Переключение сервисов (HitMoZ / SoundCloud)
- Бесконечная прокрутка результатов поиска
- Визуализатор, перемотка, громкость, перемешивание, повтор
- Сплеш-экран с анимацией волны

## Запуск

```bash
npm install
npm start
```

## Сборка установщика

```bash
npm run build
```

MSI-файл появится в папке `dist/`.

## Структура

- `main.js` — главный процесс Electron
- `preload.js` — контекстный бридж
- `index.html` — разметка UI
- `styles.css` — стили
- `app.js` — фронтенд-логика
- `wave.js` — анимация сплеш-экрана
- `hls.min.js` — HLS.js для SoundCloud-стримов
- `icon.ico` — иконка приложения
